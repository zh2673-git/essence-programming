## 数据流图

### 主数据流

```
┌─────────────┐     ┌─────────────┐     ┌─────────────┐     ┌─────────────┐
│  数据源文件  │ ──→ │  DataModel  │ ──→ │  DataFrame  │ ──→ │  Analyzer   │
│ (Excel/CSV) │     │   .load()   │     │  (pandas)   │     │  .analyze() │
└─────────────┘     └─────────────┘     └─────────────┘     └──────┬──────┘

```

## 依赖关系图

```
main.py
    ├──→ interface.CLIInterface
    │       ├──→ datasource.*Model
    │       ├──→ analysis.*Analyzer
    │       └──→ export.*Exporter
    │
    └──→ utils.setup_logger

interface/
    ├──→ datasource (数据加载)
    ├──→ analysis (执行分析)
    ├──→ export (生成报告)
    └──→ utils/config (配置读取)

export/
    ├──→ analysis (获取分析结果)
    ├──→ datasource (获取原始数据)
    └──→ utils (工具函数)

analysis/
    ├──→ datasource (获取数据)
    ├──→ base.BaseAnalyzer (继承)
    └──→ utils (存取现识别、资金追踪)

datasource/
    ├──→ base.BaseModel (继承)
    └──→ utils/config (字段映射)

utils/
    └──→ config (配置读取)
```

## 关键技术决策

| 决策 | 选择 | 理由 |
|-----|------|------|
| 架构模式 | 分层架构 | 职责清晰，易于维护和扩展 |
| 数据处理 | pandas | Python数据分析标准库 |
| 配置管理 | JSON | 灵活配置，无需修改代码 |
| 缓存机制 | 文件缓存 | 热重载支持，提升性能 |
| 报告格式 | Excel + Word | 满足专业报告需求 |
